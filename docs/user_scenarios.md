# 마이링크 (MyLink) 사용자 시나리오

서비스의 모든 핵심 기능 흐름을 도출하기 위해 `"방문자/소유자는 ____하기 위해 ____한다"` 형식으로 작성되었습니다.

## 1. 방문자 (Visitor) 시나리오
> 마이링크 페이지 방문 및 행동 경험

- 방문자는 해당 마이링크 페이지의 소유자가 누구인지 파악하기 위해 **가장 상단에 노출된 프로필(이미지, 닉네임, 소개글) 영역을 확인한다.**
- 방문자는 소유자의 추천 콘텐츠 및 주요 외부 사이트 정보를 파악하기 위해 **본문 화면에 나열된 전체 링크 목록을 위아래로 훑어본다.**
- 방문자는 본인이 원하는 특정 웹사이트나 소셜 미디어로 이동하기 위해 **관심 있는 링크의 버튼을 클릭한다.** (이때 해당 링크의 클릭 통계 테이블 수치가 +1 카운트된다.)

## 2. 소유자 (Owner) 시나리오
> 링크 및 프로필 관리를 위한 어드민 페이지 시나리오

### 2-1. 사용자 인증 (가입/로그인)
- 소유자는 번거로운 이메일 및 비밀번호 입력 절차 없이 빠르게 서비스를 시작하기 위해 **구글 소셜 로그인 버튼을 클릭하여 인증 절차를 완료한다.**

### 2-2. 프로필 관리
- 소유자는 방문자들에게 보여줄 자신만의 활동명을 변경하기 위해 **화면 상단의 닉네임 텍스트 부분을 직접 클릭하여 인라인 편집으로 닉네임을 수정한다.**
- 소유자는 본인이 어떤 사람인지 해시태그나 직군 등으로 소개하기 위해 **짧은 소개글 텍스트 영역을 직접 클릭하여 인라인 편집으로 문구를 수정한다.**

### 2-3. 링크 관리
- 소유자는 현재 자신이 방문자에게 노출하고 있는 외부 링크 전체 목록을 한눈에 파악하기 위해 **로그인 후 연결되는 어드민 페이지(대시보드)를 조회한다.**
- 소유자는 방문자에게 새롭게 공유하고 싶은 외부 외부 링크(예: 이력서 사이트)를 추가하기 위해 **'새 링크 추가' 버튼을 눌러 목적지의 URL 주소와 제목(타이틀)을 입력한다.** (추가 시 구글 파비콘 API를 통해 URL의 파비콘이 자동 적용된다.)
- 소유자는 기존에 등록한 링크의 제목 텍스트를 더욱 직관적이거나 최신 메시지로 변경하기 위해 **해당 링크 아이템의 '제목' 영역 텍스트를 클릭하여 인라인 편집으로 바로 내용을 수정한다.**
- 소유자는 기존에 등록된 해당 링크의 목적지 사이트 주소가 변경되었을 때 이를 새로고침하기 위해 **해당 링크 아이템의 'URL 주소' 영역 텍스트를 클릭하여 인라인 편집으로 타겟 주소를 수정한다.**
- 소유자는 오타가 났거나 일시적인 프로모션 등 더 이상 방문자들에게 노출을 원하지 않는 링크를 제거하기 위해 **해당 링크의 '삭제' 액션 아이콘을 클릭해 목록에서 아예 삭제한다.**

### 2-4. 통계 관리 (추후 구현)
- 소유자는 방문자들의 관심도 및 트래픽 유입률 대비 성과를 측정하기 위해 **대시보드상 링크 목록에서 각 링크에 표시된 '클릭 조회수' 수치를 확인한다.**
